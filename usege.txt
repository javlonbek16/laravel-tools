## Installation va ishlatish

1. **Package'ni o'rnatish:**
```bash
composer require yourname/laravel-tools
```

2. **Config faylini publish qilish:**
```bash
php artisan vendor:publish --provider="YourName\LaravelTools\LaravelToolServiceProvider" --tag="config"
```

3. **Comandalarni ishlatish:**
```bash
# Asosiy struktura yaratish
php artisan make:structure User

# Custom path bilan
php artisan make:structure User --controller-path=Http/Controllers/Api

# Interface'dan repository'ga sync qilish
php artisan sync:interfaces User

# Controller'dan route'larga sync qilish
php artisan sync:routes User
